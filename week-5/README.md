# Week 5

This week contains React application development using Vite.
The focus is on building modern frontend apps with React components, project structure, and Vite tooling.

## Contents

- `react-app-1/`: A React application scaffolded with Vite.
- `react-app-2/`: A second React application scaffolded with Vite.

## What I did this week

- Learned React app structure and how Vite scaffolds a frontend project.
- Built component-based frontend logic in `react-app-1/` and `react-app-2/`.
- Practiced using JSX, CSS modules, and asset management in React.
- Explored project configuration in `package.json`, `vite.config.js`, and `eslint.config.js`.

## Skills practiced

- React component creation and JSX syntax
- Vite-based frontend project setup and development workflow
- Organizing React application files and using public/static assets
- Working with React styles, routing, and stateful components (where included)
