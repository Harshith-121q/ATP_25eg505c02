# Week 5

This week contains React application development using Vite.
The focus is on building modern frontend apps with React components, project structure, and routing.

## Contents

- `react-app-1/`: A React application scaffolded with Vite and a component-based UI.
- `react-app-2/`: A React app using client-side routing and page navigation.

## What I did this week

- Learned how Vite scaffolds a React project and how to run the app with `npm run dev`.
- Built component-based UIs in `react-app-1/`, including a user list, navbar, footer, and demo components for forms and APIs.
- Created a routed React app in `react-app-2/` with pages for Home, Register, Login, and nested technology routes.
- Practiced handling component layout, importing assets, and using React Router for navigation.
- Reviewed React project configuration in `package.json`, `vite.config.js`, and `eslint.config.js`.

## Code highlights

- `react-app-1/src/App.jsx`: renders `Navbar`, `UserLists`, and `Footer`, with optional demo components for API calls and form handling.
- `react-app-1/src/components/UserList_Folder/`: builds a user listing interface with `Navbar`, `Footer`, `User`, and `UserList` components.
- `react-app-1/src/components/FormDemo.jsx`: practice form state management in React.
- `react-app-1/src/components/ApiDemo.jsx`: practice making API-like requests and handling responses.
- `react-app-2/src/App.jsx`: sets up `react-router` with nested routes for technology pages and a shared `RootLayout`.
- `react-app-2/src/components/`: includes layout components and pages for `Home`, `Register`, `Login`, `Technologies`, `Java`, `Nodejs`, and `Vue`.

## Skills practiced

- React component composition and JSX structure
- Using React Router for page navigation and nested routes
- Handling UI state, forms, and component properties
- Structuring React apps with Vite, ESLint, and shared assets
