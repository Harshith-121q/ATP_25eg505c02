# Week 1 - Day 1

This folder contains basic JavaScript assignments for practicing functions and arrays.

## Files

- `Compare#nos.js`: Compare two numbers and determine which is larger.
- `Compare2Nos.js`: Another number comparison exercise.
- `FindSmallestNoInArray.js`: Find the smallest number in an array.
- `FunsReceivesArryAndRetturnSum.js`: Function receives an array and returns the sum.
- `FunsReturnBig.js`: Function to return the largest value.
- `lastAssRetuenIndexOfanArgg.js`: Practice returning the last index of an argument.
- `SumOfArray.js`: Sum the values in an array.

## Skills practiced

- Function definition and return values
- Array processing
- Basic arithmetic and comparisons
