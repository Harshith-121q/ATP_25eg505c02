# Week 1

This week focuses on strengthening JavaScript fundamentals through algorithm practice and working with arrays, functions, and objects.
The exercises are designed to improve problem-solving, writing reusable code, and understanding how data flows through basic programs.

## What I did this week

- Implemented comparison logic in `Compare#nos.js` and `Compare2Nos.js` to determine the larger of two values using `if/else` statements.
- Built search and traversal logic in `FindSmallestNoInArray.js` to identify the smallest number in an array by iterating through each element.
- Created reusable functions in `FunsReceivesArryAndRetturnSum.js` and `FunsReturnBig.js` to practice parameter handling and return values.
- Added array reduction logic in `SumOfArray.js` to calculate the total of numeric values using loops and accumulation.
- Used index lookup and search techniques in `lastAssRetuenIndexOfanArgg.js` to return the last position of a value inside an array.
- Expanded object and array manipulation skills in `day2ass.js` by editing employee records, removing entries based on criteria, and updating nested score values.

## Code highlights

- `Compare#nos.js`: compares two numbers and outputs the greater value.
- `Compare2Nos.js`: reinforces branching logic with a second comparison example.
- `FindSmallestNoInArray.js`: iterates through an array while tracking the smallest number found.
- `FunsReceivesArryAndRetturnSum.js`: defines a function that accepts an array and returns its sum.
- `FunsReturnBig.js`: practices controlling return values with conditional logic.
- `SumOfArray.js`: demonstrates how to accumulate values in a loop.
- `lastAssRetuenIndexOfanArgg.js`: finds the last occurrence index for a given item in an array.
- `day2ass.js`: handles object updates, element removal, and nested field modification in an employee dataset.

## Skills practiced

- JavaScript fundamentals: variables, conditionals, loops, functions, and arrays
