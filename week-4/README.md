# Week 4

This week is focused on HTML structure, CSS styling, and page layout techniques.
The exercises cover standalone HTML examples, content nesting, flexbox and grid layouts, form styling, positioning, anchor navigation, and component-based page design.

## Week-4 structure

This folder includes root HTML examples plus layout subfolders.

- Root HTML files:
  - `ElementsQ1.html`: basic HTML elements and content structure.
  - `NestedEmlements.html`: nested HTML sections, lists, and element hierarchy.
  - `ex3.html` and `ex4.html`: additional page examples for HTML practice and layout exercises.

- `Anchor_assignment_1/`: anchor navigation with multiple city pages.
  - `cities.html`: main page linking to city detail pages.
  - `banglore.html`, `chennai.html`, `hyd.html`: individual city pages.
  - `style.css`: styling for anchor links, page layout, and navigation appearance.

- `flex/`: flexbox layout example.
  - `index.html`: flexbox container used for arranging content in rows or columns.
  - `style.css`: flex properties, alignment, spacing, and responsive behavior.

- `grid/`: grid layout practice.
  - `index.html`: page demonstrating CSS grid sections and placement.
  - `style.css`: grid template, gaps, and alignment styling.

- `form/`: HTML form styling.
  - `index.html`: form inputs, labels, buttons, and structured input groups.
  - `style.css`: form layout, input styling, and visual spacing.

- `table/`: table layout and styling.
  - `index.html`: HTML table with headers, rows, and data cells.
  - `style.css`: table borders, spacing, and visual formatting.

- `id/`: personal card example.
  - `IDcARD.HTML`: styled ID card layout with personal details.
  - `style2.css`: card styling, typography, and layout settings.

- `pageusingFlex/`: page layout using flexbox.
  - `index.html`: page sections arranged with flexbox.
  - `style.css`: flex container settings, spacing, and responsive design.

- `positions/`: CSS positioning examples.
  - `index.html`: positioned elements using relative and absolute placement.
  - `style.css`: position properties, offsets, and decorative layout styling.

- `front_assessgnment/flex/`: additional flex layout assessment.
  - `index.html`: page layout built with flexbox elements.
  - `style.css`: flex styling, card arrangement, and spacing.
