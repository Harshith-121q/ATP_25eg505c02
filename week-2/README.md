# Week 2

This week expands JavaScript skills into frontend components, asynchronous behavior, and backend API basics.
Work is organized into frontend mini-projects, intermediate exercises, and a Node.js backend example.

## Contents

- `day_1/`: Frontend project practice with E-commerce and task management examples.
- `day_2/`: JavaScript exercises exploring async/sync flow, object arrays, and real-world simulations.
- `day_3/`: A backend API project using Node.js and Express.

## What I did this week

- Built a small E-commerce flow in `E_commerse/`, including product listing, cart operations, stock checking, and checkout with discount logic.
- Created a task management module in `Tack_Management/` with title, priority, and due date validation, plus task completion behavior.
- Practiced asynchronous code in `AsynchronousTimeOutFunction.js` and compared it with synchronous flow in `SynchronousTimeOutFunction.js`.
- Implemented real-world examples such as bank transaction analysis, movie streaming platform ideas, and student performance reporting.
- Learned object and array copy behavior in `CopyArray.js` and `CopyObj.js`, and reviewed ES6 class syntax in `ClassEX.js`.
- Explored a simple backend server in `BackendDev/` with middleware logging and separate routes for users, products, and students.

## Code highlights

- `E_commerse/app.js`: orchestrates product browsing, search, cart additions, quantity updates, removal, and checkout.
- `E_commerse/cart.js`: manages cart items, adds products, checks stock, and removes items.
- `E_commerse/product.js`: defines product data, stock checking, and product retrieval functions.
- `Tack_Management/task.js`: task creation, completion, and listing using validation helpers.
- `Tack_Management/validator.js`: validates task title length, priority values, and future due dates.
- `BackendDev/server.js`: Express server setup with JSON parsing, middleware logging, and path-level routers.
- `BackendDev/API/`: route modules for users, products, and student data operations.

## Skills practiced

- DOM-style JavaScript application structure and logic separation
- Validation functions and user input handling
- Asynchronous programming with callbacks and timers
- Node.js backend design using Express and route modules
- Working with arrays, objects, and real-world program simulations
