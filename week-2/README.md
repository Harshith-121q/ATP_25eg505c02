# Week 2

This week expands JavaScript skills into frontend components, asynchronous behavior, and simple backend API development.
Work is organized into frontend mini-projects, practice exercises, and a Node.js backend example.

## Contents

- `day_1/`: Frontend project practice with E-commerce and task management examples.
- `day_2/`: JavaScript exercises for asynchronous programming, objects, arrays, and real-world simulations.
- `day_3/`: A backend API project using Node.js and Express.

## What I did this week

- Built a small e-commerce frontend structure in `E_commerse/` with cart, payment, product, and discount logic.
- Created a task management frontend application in `Tack_Management/` with task creation and validation.
- Practiced asynchronous and synchronous code in `AsynchronousTimeOutFunction.js` and `SynchronousTimeOutFunction.js`.
- Implemented real-world simulations such as bank transaction analysis and a movie streaming platform.
- Worked with arrays, objects, and class examples in `BankTransactionAnalyzer.js`, `CopyArray.js`, `CopyObj.js`, `ClassEX.js`, and `StudentPerformance.js`.
- Added shopping cart summary logic in `ShoppingCartSummaery.js` and learned backend routing in the `BackendDev/` API project.

## Skills practiced

- Frontend JavaScript application structure and project organization
- Asynchronous vs synchronous behavior and timeout handling
- Object and array manipulation, data processing, and business logic
- Basic backend API design, server routing, and Express setup
