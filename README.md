# ATP Assignment Summary

This repository contains weekly assignments for the ATP course, structured by weekly learning goals and practical projects.
Each week folder demonstrates what was learned and built during that week, from JavaScript fundamentals to full-stack applications.

## Weekly learning highlights

- **week-1**: JavaScript fundamentals with function practice, array operations, and simple algorithm assignments.
- **week-2**: Expanded JavaScript practice into frontend mini-projects, asynchronous programming, and a backend API basics introduction.
- **week-3**: Backend database work with MongoDB and Node.js, building models and API routes for data-driven applications.
- **week-4**: HTML and CSS layout practice, including basic page structure, flexbox, grid, forms, and responsive design examples.
- **week-5**: React application development using Vite, building modern frontend projects with React components.
- **week-6**: Full-stack employee management project with separate backend and frontend implementations.
- **week-7**: Full-stack blog application with a backend API server and React frontend.

## How to use this repository

- Start with this top-level `README.md` for the overall course summary.
- Open each `week-*` folder to read the weekly overview and see the specific assignments completed.
- Use subfolder `README.md` files for details on project setup, specific features, and implementation notes.

## Notes

- Week folders may contain additional project-specific `README.md` files inside subfolders.
- The folder names and files reflect the topics practiced each week, such as JavaScript exercises, frontend apps, and backend APIs.
