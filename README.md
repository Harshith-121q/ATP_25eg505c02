# ATP Assignment Summary

This repository contains weekly assignment work for the ATP course.
Each week folder includes JavaScript, web development, React, backend, and frontend practice projects.

## Week overview

- **week-1**: JavaScript fundamentals and basic algorithm practice.
- **week-2**: Frontend practice, asynchronous JavaScript, backend API basics, and task management features.
- **week-3**: MongoDB practice and backend database project structure.
- **week-4**: HTML/CSS layout projects with examples for elements, flexbox, grid, forms, and page structure.
- **week-5**: React + Vite applications for frontend development.
- **week-6**: Employee management project with Backend and Frontend components.
- **week-7**: Blog application with separate Backend and Frontend projects.

## How to use this repository

- Start from the top-level `README.md` for a week-by-week summary.
- Open each `week-*` folder to find a `README.md` with more details.
- Subfolders often contain project-specific README files for E-commerce, task management, backend APIs, and React apps.

## Notes

- Some subfolders already include their own `README.md` files for project-specific instructions.
- Use the folder summaries to quickly locate assignments and review how the projects are organized.
